Unlike [Flat Object](flat-object.html), Complex Object can contain nested array and objects along with primitive values.

## Complex Object Example

```json
{
    "param1": "value1",
    "param2": {
        "param3": {
            "param4": "value2"
        },
        "param5": "value3"
    }
}
```
